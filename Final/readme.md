Quickstart

# Clone and enter the finished build
git clone https://github.com/Code-Monster892/Knee-Deep-In-Silicon.git
cd Knee-Deep-In-Silicon/Final

# Build the Verilator simulation
make

# Run it
./obj_dir/Vcpu

Prerequisites

You'll need: verilator, riscv64-unknown-elf-gcc (or equivalent RV32IM-targeting toolchain), SDL2, picoloibc
